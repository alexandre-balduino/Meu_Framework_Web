
# 🚀 Meu Framework Web Python (ASGI)

Um framework web assíncrono, modular e leve, construído do zero para rodar em qualquer lugar, inclusive no **Android (Termux/PyCode)**.

## 🛠️ Tecnologias
- **Python 3.10+**
- **Jinja2** (Templates)
- **Asyncio** (Servidor Nativo)
- **ASGI Compliance** (Compatível com Uvicorn)

## 📁 Estrutura do Projeto
- `meu_framework/`: O núcleo do sistema (Roteamento, Servidor, Respostas).
- `static/`: Arquivos CSS, JS e Imagens.
- `templates/`: Arquivos HTML para renderização via Jinja2.
- `app.py`: Ponto de entrada da sua aplicação.
